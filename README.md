# Clock Widget Plugin

This project was developed as a clock widget for windows widgets. Date format uses the computer's system language

### Internal Settings

```json
{
  "clock_font_size": 64.0,
  "clock_foreground": "#FF8B0000",
  "date_font_size": 18.0,
  "date_foreground": "#FFBC8F8F"
}
```
